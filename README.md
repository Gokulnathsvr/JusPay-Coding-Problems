# JusPay Coding Problems & Solutions

## Overview
Welcome to the **JusPay Coding Problems** repository! This project is a curated collection of algorithmic solutions to the coding challenges frequently asked during the JusPay developer hiring process (such as the JusPay Hiring Hackathon). 

The goal of this repository is to document optimized, production-ready approaches to complex data structure problems, serving as both a personal archive and a resource for competitive programming prep.

## 🧠 Problem Categories Focus
JusPay technical rounds heavily emphasize specific computer science concepts. This repository includes solutions focusing on:
* **Graph Algorithms:** Shortest path, cycle detection, largest sum cycle, and maximum weight node problems.
* **Advanced Tree Structures:** N-ary trees, particularly the famous "Tree of Space" (locking, unlocking, and upgrading nodes).
* **Concurrency & Multi-threading:** Thread-safe operations on data structures.
* **Optimization:** Focus on time complexity (O(1) or O(log N) operations where possible) and space efficiency.
